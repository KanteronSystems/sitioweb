Clinical Genomics solution

Si su sistema actual de PACS-VNA-EHR no soporta datos de Genómica, Kanteron Systems le propone emplear nuestro módulo que dispone de:


Un navegador de genoma en 2D y en 3D via web de nueva generación

  • Navegación en tiempo real a través de los cromosomas y los cariotipos, representando diferente tipos de datos de muchos tipos de información genómica
  • Los datos se representa utilizando SVG directamente del lado del cliente. No se generan imágenes en el servidor y se envían al cliente (lo que significa un servidor de datos más rápido y datos más cacheables)
  • Numerosas pistas preconfiguradas tales como los genes, transcripciones, SNPs, mutaciones, objetivos de miARN, TFBS, etc
  • Escrito en Javascript, hace uso de estándares abiertos como HTML5 o CSS3
  • Zero-footprint: funciona en cualquier navegador web moderno sin la instalación de ningún plug-in Flash , Java Applet o cualquier otra cosa
  • Muy rápido, desplazamiento suave y respuesta dinámica a las peticiones del usuario
  • Utilice archivos locales o remotos, servidores DAS o cualquier otro adaptador de datos a través de servicios web
  • Importación de datos y archivos de NGS (BAM, VCF, GFF, BED…)
  • Exportar datos en formato JSON
  • Permite seleccionar áreas de la secuenciación y hacer zoom para visualizar histogramas
  • Búsqueda de genes, SNPs, variaciones, exomas y transcriptomas
  • Permite ampliar la información de una posición concreta
  • Permite cargar y trabajar con cellbase y DAS
  • Historial de posiciones revisadas
  • Descargar a secuenciación en raw (.ban) para verlo en el visor de preferencia
